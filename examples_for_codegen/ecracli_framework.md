# Ecracli framework

## File Structure

- **Ecracli.py** – Driver file for the cli
- **clis/*.py** – Each command will be defined here along with sub-commands and operations
- **tmpl/help.json** – Help text for all the clis
- **tmpl/parameters.json** – Optional and mandatory params for each cli
- **User_manual.md** – Manual for the clis

## Ecraapi.yaml/exacc-ecraapi.yaml
